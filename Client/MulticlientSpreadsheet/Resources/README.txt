The cells can be selected by mouse clicks or by using the arrow keys. 
My spreadsheet will always display the content of the cell you are on. 
I have message box for all the exceptions that happen and revert back 
to the state before the exception was thrown. 

Extra Features

1) Can use arrow keys to move 
2) Copy and Paste fuctionality
3) Hot Keys for all menu options
4) Can open a spreadsheet in a new window or current window. 
5) Can right click to show options to empty the current spreadsheet.
